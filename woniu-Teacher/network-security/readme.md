# 网络安全
常见web攻击方式
 - XSS
    Cross-Site Scripting (跨站脚本攻击，因为缩写和css重叠所以叫XSS，跨站脚本攻击是指通过存在安全漏洞的web网站注册用户的浏览器内运行非法的非本站的HTML标签或JavaScript进行的一种攻击)

    跨站脚本攻击有可能造成以下影响:
      1. 利用虚假输入表单骗取用户信息
      2. 利用脚本窃取用户的Cookies值，帮助攻击者发送恶意请求
      3. 显示伪造的文章或图片


 - CSRF
 - 点击劫持
 - SQL注入
 - OS注入
 - 请求劫持
 - DDOS

 防范方法
  - 密码安全 - 密码加固
  - 密码学 (对称与非对称)
  - 传输安全 - HTTPS
  - NOdeJS安全框架 hemelt
  - CSP策略
