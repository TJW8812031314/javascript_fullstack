#  预编译 四部曲 发生在函数执行之前
1. 创建一个AO对象
2. 找形参和变量声明， 将变量声明和形参作为AO属性名，值作为undefined
3. 将实参值和形参值统一
4. 在函数体里面找函数声明，将函数名作为AO对象的属性名， 值赋予函数体

AO {
  <!-- a: undefined, -->
  <!-- a: 1, -->
  <!-- a: function () {}, -->
  a: 123
  <!-- b: undefined, -->
  b: function () {}
  d: function () {}
}

# 预编译也发生在全局
1. 创建GO对象
2. 找形参和变量声明， 将变量声明和形参作为GO属性名，值作为undefined
3. 在函数体里面找函数声明，将函数名作为GO对象的属性名， 值赋予函数体


- 任何变量未声明就赋值， 此变量就会是全局变量
- 一切声明的全局变量，都window所有

- 函数声明整体变量提升
- 变量   声明提升
